My Project Portfolio

By: Al Rey Villagracia

Data Engineering & Pipelines

  • Designed and built a full pipeline simulator in Streamlit integrating DBT-style SQL, REST ingestion, and lineage DAG for cloud data operations.
  • Developed a rule-based faculty-course assignment engine in Python enforcing over 10 constraints using pandas and Excel inputs.

Dashboards & KPI Reporting

  • Created an interactive dashboard in Streamlit to visualize Microsoft 365 digital adoption KPIs, usage funnels, and user trends.
  • Analyzed national Social Health Development Index (SHDI) data across multiple years using pandas and seaborn to produce data-driven insights.

Machine Learning & AI

  • Built LightGBM regression models using RDKit molecular descriptors to predict 5 polymer properties from SMILES input; evaluated using custom wMAE.
  • Implemented Q-learning and PPO agents with Stable-Baselines3 to match supply with dynamic demand, benchmarking against LP optimization.

Document & Semantic Search

  • Engineered a Streamlit app that parses DOCX resumes, extracts key skills, calculates job match score, and supports DOCX/Markdown export.
  • Developed a RAG (Retrieval-Augmented Generation) pipeline for question-answering over enterprise docs using FAISS, LangChain, and OpenAI APIs.

Security & Compliance

  • Designed and implemented a secure, HIPAA/GDPR-compliant patient data system using MySQL with AES encryption and RBAC controls.

Research & Simulation

  • Simulated A/B testing under network interference using cluster randomization and causal inference graphs in Python.
  • Led strategic market entry simulation including Porter’s Five Forces, product team structure, and pitch preparation for Cam Car product case.

Geospatial & Disaster Resilience

  • Constructed a climate resilience index from uploaded regional data using geopandas and Streamlit with interactive mapping and CSV parsing.

Education & Evaluation Tools

  • Built a Streamlit tool to evaluate demo lessons with rubric scoring, signature capture, and multi-form PDF export functionality.

CRM & Workflow Automation

  • Designed an SLA-aware ticket routing prototype using Power Apps, Dataverse, and Power Automate to streamline support workflows.
